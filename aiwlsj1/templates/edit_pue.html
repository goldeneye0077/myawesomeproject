    关于PUE

    PUE（Power Usage Effectiveness）是衡量数据中心能源效率的指标：

    • PUE = 总耗电量 / IT设备耗电量
    • 优秀： PUE ≤ 1.3
    • 良好： 1.3 < PUE ≤ 1.8
    • 一般： 1.8 < PUE ≤ 2.0
    • 较差： PUE > 2.0

    理想情况下PUE值越接近1.0越好，表示能源利用效率越高。
